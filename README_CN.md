### 介绍

一共有两个页面，分别为 Splash 和 Home 页面。Splash 页面主要为一段介绍和一个进入按钮，Home 页面主要分为：菜单、信息、分类、最喜欢以及周边餐馆，喜欢模块和附近餐馆包含评分和价格以及到达时间等。
